# Wifey Money — Liquid Glass Local

A local-only Poppins mobile web app for tracking money temporarily kept in your bank account.

## Pages

- Home: quick balance dashboard
- Add: fast transaction entry
- History: search, filter, delete, undo
- Backup: export/import JSON and CSV
- Settings: currency, app name, theme, clear data

## Privacy

This version is local/backup only.

Records are saved in the browser using localStorage. The hosted files can be public, but your money data stays on the device unless you export/import backup files.

Do not upload private backup JSON files to GitHub.
